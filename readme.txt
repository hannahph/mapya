MapYa simplifies planning your day while traveling by aggregating all of your searches into a single map-based view. For more information, watch our presentation: https://www.youtube.com/watch?v=ug6gKCbBffA&feature=youtu.be

You can run MapYa 2 ways:

1) Unzip the termProject.zip file, navigate to the backend folder, and run "node server.js"

2) Pull down the docker image from dockerhub using "sudo docker pull hannahph/mapya:final" and then run the docker image with "sudo docker run -p 8080:8080 hannahph/mapya:final"
